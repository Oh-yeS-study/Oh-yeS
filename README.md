## 스터디 진행방법
**_기술면접 대비 Q&A 방식_**
- 모든 스터디원이 해당 범위의 학습 분량 공부
- 각자 4개의 질문 준비
- 각자 4개의 질문을 받고, 즉석 답변하는 방식
- 스터디 종료 후, 각자 받은 질문을 정리하여 포맷에 맞춰 Github에 정리
  - `질문 - 답변` 형태로 정리
